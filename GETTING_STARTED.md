# 🚀 Getting Started - Wistech Solutions Homepage

Welcome! Your professional homepage is ready. Here's how to get started in 5 minutes.

## ⚡ Quick Start (5 Minutes)

### Step 1: View Your Website (1 minute)
1. Find `index.html` in your project folder
2. Double-click to open in your web browser
3. Review all sections from top to bottom
4. Test on mobile by resizing your browser window

### Step 2: Customize Company Name (2 minutes)
1. Open `index.html` with a text editor
2. Press `Ctrl+H` to open Find & Replace
3. Search: `Wistech`
4. Replace with: `Your Company Name`
5. Save the file
6. Refresh your browser to see changes

### Step 3: Update Contact Information (2 minutes)
1. Open `index.html` with text editor
2. Press `Ctrl+F` to find: `contact@wistechsolutions.com`
3. Replace with: `your-email@company.com`
4. Find: `+1 (234) 567-890`
5. Replace with: `your actual phone`
6. Save and refresh browser

✅ **Done!** Your basic website is ready.

## 🎨 Customization Examples

### Change Hero Title
In `index.html`, find:
```html
<h1 class="hero-title">Innovative IT Solutions for Modern Businesses</h1>
```
Replace with:
```html
<h1 class="hero-title">Your New Headline Here</h1>
```

### Change Primary Color
In `styles.css`, find:
```css
--primary-color: #1e3c72;
```
Replace `#1e3c72` with your color. Examples:
- Red: `#FF0000`
- Green: `#00AA00`
- Purple: `#8B00FF`

### Update Service Titles
In `index.html`, find services section and update:
```html
<h3>Web Development</h3>
```
to:
```html
<h3>Your Service Name</h3>
```

## 🆘 Common Issues

### Issue: Styles not loading
**Fix**: Make sure `styles.css` is in the same folder as `index.html`

### Issue: Site looks broken on mobile
**Fix**: Your browser may be zoomed. Set zoom to 100% (Ctrl+0)

### Issue: Changes not showing
**Fix**: Hard refresh browser (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)

### Issue: Menu doesn't work
**Fix**: Check JavaScript is enabled in browser settings

### Issue: Colors look wrong
**Fix**: Check your monitor display settings or try different browser
